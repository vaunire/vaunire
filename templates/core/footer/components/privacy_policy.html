Политика обработки персональных данных

Последнее обновление: 18 декабря 2025

1. Введение

Мы уважаем вашу конфиденциальность. Данная политика описывает, как сайт «Vaunire» (Учебный проект) обрабатывает информацию. Посещая этот сайт, вы соглашаетесь с тем, что вводимые данные используются только для демонстрации работы программного кода и не имеют юридической силы реального договора.

2. Сбор тестовых данных

В рамках тестирования функционала сайта (формы заказа, регистрации, подписки) мы можем технически обрабатывать следующие данные, которые вы вводите добровольно:

  • Имя и фамилия (для имитации оформления заказа);
  • Номер телефона (для проверки работы маски ввода JS);
  • Адрес электронной почты (для имитации почтовых рассылок Django).
Важно: Пожалуйста, не вводите реальные паспортные данные, номера действующих кредитных карт, пароли от важных аккаунтов. Используйте вымышленные данные для тестирования.

3. Цели обработки

Единственной целью сбора данных является тестирование backend-части приложения (Django), проверка корректности работы базы данных (PostgreSQL) и валидации форм на стороне сервера.

4. Хранение и передача данных

Данный сайт является демонстрационным стендом. Мы не передаем данные третьим лицам, не продаем их и не используем для маркетинга. База данных проекта регулярно очищается.
